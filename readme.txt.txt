Community Science Museum
Description
This project is a responsive and visually appealing website for a Community Science Museum. It showcases interactive elements, a user-friendly sidebar, and social icons for better engagement.

Features
HTML: For the website's structure.
CSS: For styling and design.
JavaScript: Used for the sidebar functionality.
Images: Optimized images, all under 200KB, for fast loading.
Font Awesome: Social media icons integration for connectivity.
Responsive Design: Created using media queries to ensure the website works seamlessly on all screen sizes.
Technologies Used
HTML
CSS
JavaScript
Font Awesome

Open the project folder.
Run the index.html file in any modern web browser.
Media Queries
This project employs media queries for a responsive layout, ensuring the website is attractive and functional on desktops, tablets, and mobile devices.
